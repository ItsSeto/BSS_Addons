## BSS-Addons

BSS Addons is a program designed to make Roblox Bee Swarm Simulator easier to play, brings up with many QoL features to help with the game itself!

- Passive Counter : Tracks the count of Passive Ability, no more missing the line up while Boosting!
- AutoItems : Automatically uses Items with customizable Hotkeys, Item slots and Cooldown.
- And many more!(Still working on)
